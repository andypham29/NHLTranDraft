# NHLTranDraft
Web App Fantasy Draft

## Installation

1. Install Nodejs: https://nodejs.org/en/download/
2. Install Heroku-Cli: https://devcenter.heroku.com/articles/heroku-cli


## Usage

1. npm install
2. npm start || heroku local
3. open browser to localhost:5000
